9. Micrositio o Sitio de Datos Abiertos
=======================================

Ingrese a la URL http://microsites.dev:8080/home


.. image:: ../_static/images/image19.png

El sitio muestra un icono de la página web, título de bienvenida, breve introducción de lo que trata el micrositio, catálogo de recursos publicados, filtro para la búsqueda de recursos, tipos recursos, categorías principales de los recursos y referencias.

.. image:: ../_static/images/image17.png

Para acceder a los recursos del sitio de datos, clic sobre el recurso requerido.

Conjunto de Datos |image12|: El sitio visualiza el Conjunto de Datos con sus recursos asociados (si lo tiene) y un buscador de recursos. También la opción de Compartir es una posibilidad que da Junar para mostrar a otros los recursos de tu interés de una manera sencilla. Podrás compartir vía Twitter, Facebook, Google+, LinkedIn |image11|.



En el margen izquierdo de la visualización del Conjunto de Datos se muestran los siguientes botones:

+ Para acceder a la información del Conjunto de Datos presione |image14|
+ Para descargar el archivo del Conjunto de Datos presione |image13|

  .. image:: ../_static/images/image10.png

Vista |image06|: El sitio de datos visualiza el contenido de la Vista, un buscador de recursos y permite compartir la Vista de manera sencilla a través de Twitter, Facebook, Google+, LinkedIn |image11|.

En el margen izquierdo de la visualización de la Vista se muestran los siguientes botones:


+ Para acceder a la información de la Vista presione |image14|
+ Para actualizar el contenido de la Vista presione |image68|
+ Para insertar la Vista presione |image70|
+ Para solicitar la Vista de datos a través de la API presione |image69|
+ Para exportar la Vista de datos en formato CSV, XLS/XLSx o usar Google Spreadsheet presione |image66|


  .. image:: ../_static/images/image65.png


Visualización |image67|: El sitio de datos muestra la Visualización, un buscador de recursos y permite compartir la Visualización de manera sencilla a través de Twitter, Facebook, Google+, LinkedIn |image11|.

En el margen izquierdo de la Visualización se muestran los siguientes botones:

+ Para acceder a la información de la Visualización presione |image14|
+ Para actualizar el contenido de la vista presione |image68|
+ Para insertar la visualización presione |image70|
+ Para solicitar la Visualización a través de la API presione |image69|
+ Para exportar la Visualización en formato CSV o XLS/XLSx presione |image66|
